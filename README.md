# 科技小亮AGI个人主页

一个用于 GitHub Pages 的静态个人品牌主页，风格方向是“内容实验室”：暖色、线条卡通、实验记录卡片。

## 图片占位

把图片放到以下路径后，页面会自动显示真实图片；如果图片不存在，会保留手绘占位样式。

```txt

public/images/hero-lab.png
public/images/qrcode-wechat.png
public/images/qrcode-xiaohongshu.png
public/images/qrcode-douyin.png
```

## 本地预览

直接在浏览器打开 `index.html` 即可预览。

## GitHub Pages

如果使用仓库根目录部署，GitHub 仓库设置里选择：

- Source: Deploy from a branch
- Branch: `main`
- Folder: `/root`
